# Currency Converter

Real-time currency converter supporting 180+ currencies with live exchange rates.

## Live Demo
[View Project](https://reachankitv.github.io/currency-converter/)

## Features
- Convert between 180+ world currencies
- Real-time exchange rates via API
- Live conversion as you type
- Reverse conversion (type in either field)
- Currency swap button
- Clean, intuitive interface
- Error handling

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- ExchangeRate API

## How to Use
1. Enter amount in FROM field
2. Select source currency
3. Select target currency
4. Conversion happens automatically
5. Use swap button to interchange currencies
6. Click CLEAR to reset

## What I Learned
- Working with financial APIs
- Event-driven programming (input, change events)
- Bidirectional data flow
- Async/await for API calls
- Dynamic UI updates

## API Used
- ExchangeRate-API (free tier)
- Supports 180+ currencies
- Real-time exchange rates
